### Technology and Tools

- [Hardhat](https://hardhat.org/)
- [Solidity](https://soliditylang.org/)

## Get started

```sh
yarn install
```

## Local blockchain

```sh
$ npx hardhat node --fork https://mainnet.infura.io/v3/YOUR_INFURA_API_KEY
$ npx hardhat --network localhost run scripts/deploy.ts
```
